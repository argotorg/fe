Trait associated functions

This change allows trait functions that do not take a `self` parameter.
The following demonstrates a possible trait associated function and its usage:

```
trait Max {
  fn max(self) -> u8;
}

impl Max for u8 {
  fn max() -> u8 {
    return u8(255)
  }
}

contract Example {

  pub fn run_test(self) {
    assert u8::max() == 255
  }
}
```
