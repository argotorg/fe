Braces! Fe has abandoned python-style significant whitespace in favor of the
trusty curly brace.

In addition, `elif` is now spelled `else if`, and the `pass`
statement no longer exists.

Example:
```
pub struct SomeError {}

contract Foo {
  x: u8
  y: u16

  pub fn f(a: u8) -> u8 {
    if a > 10 {
      let x: u8 = 5
      return a + x
    } else if a == 0 {
      revert SomeError()
    } else {
      return a * 10
    }
  }

  pub fn noop() {}
}
```
