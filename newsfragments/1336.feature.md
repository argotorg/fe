Added PC-accurate debug information. `fe dev trace emit` compiles a
standalone Fe file or a whole ingot and writes a validated stream of typed
compiler facts
(JSONL) that links every EVM bytecode instruction back through the backend
and MIR to exact source spans, and `fe dev debug emit --format ethdebug`
projects that stream into an ethdebug instruction/source artifact with an
optional origin/confidence sidecar. External tools can now resolve a program
counter from any EVM execution trace to the Fe source expression it was
compiled from. Attribution is honest by construction: every instruction is
classified as source-mapped, ambiguous, synthetic, or unmapped, and only
mappings proven unique carry source context.
