Added support for project manifests and project dependencies.

Example:

```
my_project
├── fe.toml
└── src
    └── main.fe
```

```
# fe.toml
name = "my_project"
version = "1.0"

[dependencies]
my_lib = { path = "../path/to/my_lib", version = "1.0" }
my_other_lib = "../path/to/my_other_lib"
```

Note: The current implementation supports circular dependencies.
